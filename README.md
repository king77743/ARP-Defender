# ARP-Defender 🛡️

ARP-Defender — это легковесный и мощный инструмент на Python для обнаружения и активного предотвращения атак типа ARP-spoofing (MITM). 

Скрипт не просто уведомляет об атаке, он активно восстанавливает сетевые таблицы жертвы и роутера, "выбивая" хакера из канала связи.

---

## 🚀 Особенности

*   Passive Monitoring: Постоянное сканирование сети на предмет подозрительных ARP-ответов.
*   Active Defense: Автоматическая отправка "лечебных" пакетов (ARP-резолвинг) для восстановления связи.
*   Visual Alerts: Яркая цветовая индикация атак в терминале для мгновенного обнаружения угрозы.
*   Double-Sided Protection: Исправляет ARP-таблицы как у целевого устройства, так и у шлюза (роутера).
*   Stability: Полная обработка исключений (KeyboardInterrupt, PermissionError).

## 🛠 Требования

*   Python 3.13.4
*   Scapy библиотека
*   Linux OS (рекомендуется Kali Linux или Ubuntu)
*   Права суперпользователя (root)

## 📦 Установка

1. Склонируйте репозиторий:
*   git clone https://github.com/king77743/ARP-Defender
*   cd ARP-Defender
2. Установите зависимости:
  ```
pip install scapy
   ```
## 💻 Использование

Для работы с низкоуровневыми пакетами скрипту нужны права sudo:
     
   ```
   sudo python3 arp-defender.py
   ```

Как это работает:
  1. Вы вводите IP вашего роутера
  2. скрипт получает настоящий MAC-адрес роутера
  3. Система переходит в режим охраны. Как только хакер попытается подменить MAC роутера на свой, скрипт выведет алерты и начнет контр-атаку "пакетами правды", восстанавливая интернет у жертвы.
